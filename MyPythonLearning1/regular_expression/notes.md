# 正则表达式的备忘录
对于 Python 来说，不学**正则表达式**感觉是等于白学 Python。

正则表达式最重要的作用就是匹配字符。下面来详细介绍一下用法和语法。

## 元字符
元字符的作用在于 “选择模式” ，就正如我们在使用 Word 或者 VSCode 时，会选择 “全字匹配” 或者 “大小写敏感” 模式一样。

元字符包括如下的内容：
```
. * + ? \ [ ] ^ $ { } | ( )
```
### 字符 `.` 的作用
`.`表示匹配指定字符和其向前或向后**一个**字符，不换行。例如：
```
中国智造，惠及全球
```
我们键入表达式 **`.造`**，则将匹配 “智造”，同时键入 **`智.`** 也将获得同样的结果。注意键入尾部字符 `球.` 则报错，因为 “不换行” 政策。

正则表达式在 Python 中代码的写法：
```ruby
content = '''苹果是绿色的
橙子是橙色的
香蕉是黄色的
乌鸦是黑色的'''

import re
p = re.compile(r'.色')
for one in  p.findall(content):
    print(one) 
```
输出为：
```
绿色
橙色
黄色
黑色
```

### 字符 `*` 的作用
`*` 表示对指定字符匹配任意次，同时在指定字符串**结尾**的字符允许出现多次（或**不存在**）来匹配。例如键入 **`绿色*`** 匹配如下内容：
```
苹果是绿色色色色的
橙子是橙色的
香蕉是黄色的
乌鸦是黑色的
树叶是绿色的
树叶是绿的
```
将匹配为：
```
绿色色色色
绿色
绿
```

<br>如果与 `.` 结合为 `.*`，则表示对**指定字符以及向后的所有字符**匹配（仅对单行有效），而 `*.` 这种用法不存在。上述内容使用指令 **`是.*`** 将匹配为：
```
是绿色色色色的
是橙色的
是黄色的
是黑色的
是绿色的
是绿的
```

<br>在前面我们提到对于行尾字符我们再去加 `.` 匹配会报错，如果想要不报错，可以尝试 **`球.*`**，因为 `*` 表示的任意次可以是 0 次。

### 字符 `+` 的作用
`+` 表示对指定字符匹配任意次，但在指定字符串**结尾**的字符必须出现至少一次才可以匹配。例如对上述例子，`绿` 将不会被指令 **`绿色+`** 匹配。

下面来看一个例子：
```
树叶是绿色的
树叶是
```
输入指令 `是.+` 的匹配结果为：
```
是绿色的
```
而输入指令 `是.*` 的匹配结果为：
```
是绿色的
是
```
造成这种差异的原因是：`+` 需要检索到至少一次或一个字符，对 `.`（任意匹配后一个字符）加上 `+` 而言，意思就是匹配指定字符加后面所有的字符（至少存在一个），而 `.*` 在后面有字符就全匹配，没有也要匹配指定字符本身。
